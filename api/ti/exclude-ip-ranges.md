# TI 제외 IP 대역(사설·특수용도)

### IPv4 (RFC1918)

| CIDR           | Range                         | 설명              |
| -------------- | ----------------------------- | --------------- |
| 10.0.0.0/8     | 10.0.0.0 ~ 10.255.255.255     | 사설 IP (Class A) |
| 172.16.0.0/12  | 172.16.0.0 ~ 172.31.255.255   | 사설 IP (Class B) |
| 192.168.0.0/16 | 192.168.0.0 ~ 192.168.255.255 | 사설 IP (Class C) |

| -------- | ----------------------------------- |

## 실무 운영 메모

* 위 대역은 **TI 평판 조회를 스킵**해도 무방
* CGNAT(100.64.0.0/10)은 외부 통신에 사용되지만 **실제 사용자 식별 불가**
* 169.254.169.254 (클라우드 메타데이터 IP)는 Link-Local → TI 스킵
* Bogon 전체 리스트를 운영에 반영하면 더 안전하지만, 위 최소 세트로도 충분
